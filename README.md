- **Responsive UI**: Built with **TailwindCSS** and **DaisyUI** for a mobile-first design.

- **Forms & Validation**: Uses **React Hook Form** for smooth and scalable form handling.  
- **Media & Carousel**: Includes **Swiper** and **React Responsive Carousel** for showcasing content.  
- **Routing**: Easy navigation with **React Router**.
